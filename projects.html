WISTL Compiler

Developed a teaching compiler for the WISTL language in C, implementing a full pipeline including lexical analysis, recursive-descent parsing, semantic checks, and code generation targeting the JVM. Applied deterministic finite automata for scanning, recursive parsing for an LL(1) grammar, and symbol/type tables for semantic validation. In accordance with Stellenbosch University's Academic Integrity Policy this project can not be publicly hosted (Available on Request).

Quasi-Palindromic DNA Analysis Utility

Built a Java program to search for and generate quasi-palindromic substrings in DNA sequences under user-defined constraints (substring length, swap limits, jump ranges). Implemented efficient algorithms for palindrome checking, minimum-swap detection, and recursive sequence generation with caching, enabling scalable analysis of large DNA inputs.

Python-Based Board Game

Developed a two-player command-line board game in Python, implementing core game logic including piece movement, win conditions, and turn management. Engineered a robust system for move validation to ensure players adhered to game rules and managed the game state and scoring, with points awarded for moving pieces into designated "sinks" on the board.

Statera

Developed a comprehensive desktop application in Java using the NetBeans IDE to help users balance their professional and personal lives. Engineered a 'Focus Mode' to block distracting websites and an automated 'workaholic filter' that prompts users to take breaks after 60 minutes of work.
